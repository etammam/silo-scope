# 🌾 Siloscope

**Siloscope** is a developer utility designed to bridge the gap between your local environment and a running Orleans Cluster. It allows you to interact with Virtual Actors as easily as you would with a REST API in Postman.

### Key Features:
* **Dynamic Discovery:** Point to a local `.dll` or search NuGet to reflect Grain interfaces.
* **Cluster Agnostic:** Join clusters using Redis, ADO.NET, Azure, or Static clustering.
* **Dual-Mode:** Use the **Desktop UI** for ad-hoc testing or the **CLI** for automated workflows.
* **Zero Boilerplate:** No need to write custom client wrappers or dummy controllers just to trigger a Grain method.
